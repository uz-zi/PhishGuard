# Privacy Policy for PhishGuard

**Last Updated:** May 2026

PhishGuard is a browser extension designed to detect phishing websites in real time using machine learning.

## Data Collection

PhishGuard does **not** collect, store, sell, or share any personally identifiable information.

The extension does **not** collect:

- Names
- Email addresses
- Passwords
- Payment information
- Cookies
- Browsing history
- Personal account data

## URL Processing

To detect phishing websites, PhishGuard temporarily processes the URL of the webpage currently being visited.

The URL may be sent to the PhishGuard backend API for phishing analysis.

This URL is:

- Used only for real-time phishing detection
- Not linked to user identity
- Not permanently stored
- Not shared with third parties for advertising or tracking

## Third-Party Services

PhishGuard uses a backend API hosted on Hugging Face Spaces for machine learning inference.

## Data Security

PhishGuard is designed with privacy in mind and processes only the minimum information required for phishing detection.
